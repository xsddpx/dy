# xdy

- 将含 `AGENTS.md` 与 `DOCS/PROJECT.md` 的当前 Git 根目录设为 `PROJECT_ROOT`；找不到时使用 `~/Codex/dy`。
- 读取 `~/.codex/AGENTS.md`、`$PROJECT_ROOT/AGENTS.md` 和 `$PROJECT_ROOT/DOCS/PROJECT.md`，再按项目路由读取模块；默认读取 01–04，环境问题读 00，评论任务读 06。
- 空调用执行 `anna auto/fast`；`fast` 等同 `auto`；附加内容作为本次主题、衣柜图编号、模板或发布要求。
- 正式视频固定使用三张参考图：`@图1` 人物、`@图2` 衣柜人台商品图、`@图3` 环境；衣柜图片和服装描述必须来自模块 05 的同一入库条目。
- 不执行 MP4 技术检查；默认通过模块 02 的胸部体量内容质检后自动上传 Google Drive 根目录，再同步发布抖音和快手；用户要求发布前确认、只生成或不发布时仍完成 Drive 上传，并在平台发布前停在确认节点。
- 所有流程与验收以项目文档为准，不在本技能重复 SOP。
