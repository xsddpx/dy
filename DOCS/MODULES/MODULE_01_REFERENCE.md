# 模块 01：自主选题、衣柜匹配与 grid prompt

## 职责

- 校验并复用当前 active 一周行程；缺失、无效、未开始或过期时，先重建今天起连续 7 天行程。
- 默认流程：先自主设计本次作品的创作方案和画面方案，再从 Anna 衣柜优先匹配款式。
- 写入 `TEMP/RUN_ID/grid-prompt.txt`；后续阶段只通过 `TOOLS/prompt_lint.py derive` 从它派生 prompt。

## 默认自主分支

- 当前日期位于 active 行程窗口内时必须复用；资产缺失、无效、未开始或过期时，先生成今天起连续 7 天的国内真实城市行程并保存，再继续。
- 当天内容必须匹配行程资产中的日期、城市、季节天气依据、白天/晚上方向、活动、旅行状态、穿搭方向和主题关键词；行程地点只是大方向，不是 prompt 环境的硬锁。
- 生成或重建一周行程资产时，每天不得写具体钟点，`time_slot` 统一写 `白天；晚上`；每天必须同时包含白天和晚上两个可拍方向，分别写清活动、可见场景、穿搭方向和主题关键词。需要更细分时可使用 `daytime_option` 和 `night_option`，但二者也不得写具体钟点。
- 每次生成前必须先从 `早上`、`上午`、`下午`、`晚上` 中随机确定一个拍摄时段，并把该时段落实到拍摄语境、环境光线和画面方案里；`早上`、`上午`、`下午` 使用当天行程的白天方向，`晚上` 使用晚上方向。除用户明确指定时段外，不得因夜景灯光、玻璃反光或酒店氛围稳定而连续偏向晚上。
- 每次生成前必须从当天行程里确定一个拍摄语境或展示舞台，例如酒店窗边、商场连廊、街区灯光、海边步道、泳池外廊、餐厅门口、交通中转空间或夜间门廊；行程用于提供真实场景、季节、光线和旅行状态，不要求每条都由购物、吃饭、房卡、端咖啡、整理行李等生活动作驱动。
- 执行者先基于核心卖点、当天行程和用户当次主题，以短视频创作者视角自主确定本次作品的卖点展示结构、场景、动作、镜头、发布文案方向和互动方式；写 prompt 前先明确本条视频如何更好展示 S 型身材、穿搭轮廓、腰线、上身辨识度、臀胯比例和成熟女性妩媚感。
- 本条视频的新鲜点优先来自卖点呈现方式：侧身曲线、转身走近、腰线停顿、前后景深、镜头距离、拍摄关系、轻舞律动、试衣走位、肩颈展开、腰胯重心变化或视线调度；生活语境、手部展示和道具互动只有在能强化卖点时才作为新鲜点，不得盖过镜头展示主轴。
- 镜头设计必须优先服务 S 型身材展示、穿搭轮廓、腰线、上身辨识度和成熟曲线；拍摄关系可按当次展示设计选择自拍、固定拍摄或他人手持拍摄，不默认限定为某一种视角。镜头关系必须符合自身特性：固定拍摄的镜头位置不移动，只允许人物在画面内转身、重心变化、半步走近或姿态变化；他人手持拍摄可以有轻微稳定手持感，但主动跟拍、后退或推近必须服务人物走路、走近、走位或试衣走位；自拍只能体现人物持机带来的轻微角度变化，不写成第三人称跟拍。
- 每条内容可把一种可见才艺或表现力作为新鲜点之一，例如轻舞律动、穿搭搭配、镜前造型、轻运动/拉伸、试衣走位、手部造型、道具互动或镜头表现力；才艺必须通过动作、节奏、姿态或道具任务被看见，不写空泛能力说明。
- 自主设计后，默认按当天日期优先选择 `MATERIAL/anna-wardrobe.md` 中对应编号的衣柜款式，例如 7 号优先 `夏-07`、19 号优先 `夏-19`，让 Anna 的日常穿搭按日期自然轮换。
- 当天编号款式缺失，或明显不匹配 active 行程、场景、天气、展示舞台、动作需求和本次创作设计时，执行者可自主设计或改选可发布穿搭，但必须记录跳过日期编号款式的原因，并保持轻熟性感、修身、腰线、上身辨识度、S 型成熟曲线、成熟女性妩媚感和互动感这些核心卖点。
- 实际写入 `穿搭：` 段时，只允许提取被选中衣柜条目 `款式提示词：` 后面的内容；不得把季节标签、环境标签、编号、元数据或使用建议拼进最终 prompt。
- 拍摄语境、展示动作和现有视频类型对应即可，不新增类型名：酒店窗边、商场连廊、街区灯光、出门和交通中转可落到 `穿搭展示`、`走路回头` 或 `生活场景剧情`；跑步、拉伸、健身、泳池外廊和运动休闲通常落到 `健身运动`；镜前展示仍可落到 `镜前自拍`，但不得连续多条只做镜前摆拍。
- 行程匹配、拍摄语境选择、卖点校准、场景动作变化和衣柜取舍只用于内部判断；`grid-prompt.txt` 只写可直接生成的可见内容，不写分析字段、文件或来源、流程、合规、平台解释及“根据/融合/吸收”等表达。

## grid-prompt 固定结构

`grid-prompt.txt` 必须使用十段式结构，且段落顺序固定：

- `人物：`
- `视频类型：`
- `穿搭：`
- `姿态镜头：`
- `环境：`
- `卖点与锁定：`
- `表情节奏：`
- `整体动画：`
- `背景音乐：`
- `其他：`

每段冒号后都必须写最终可执行内容，不得保留省略号、尖括号占位、条件分支或模板说明。`人物：` 段统一写成确定身份锚定句，只写身份、五官、脸型、发型、自然神态和稳定身材比例，不写姿态、穿搭、环境或画面锚点。

`auto/fast` 视频 prompt 使用：

```text
人物：@图1 是同一位成年女性的多视角、多表情角色参考图，不是多人合照；以 @图1 中左下大脸和正面脸为主要身份依据，侧面、背面和表情小图只用于辅助保持发型、脸型、身材比例和整体气质。以 @图1 中的同一位成年女性作为身份、五官、脸型、发型、自然神态和稳定身材比例依据，保持同一人物的一致性。画面中只出现这一位成年女性。
```

## 视频类型规则

- `视频类型：` 段必须写主类型、次类型和一句短的成片方向，例如 `视频类型：穿搭展示；次类型：健身运动；近景穿搭展示结合轻运动互动节奏。`。
- 主类型只能选择一个最贴近的类型：`舞蹈律动`、`健身运动`、`穿搭展示`、`镜前自拍`、`走路回头`、`坐姿互动`、`双人互动`、`生活场景剧情`。
- 次类型可选且最多一个；需要时从同一集合中选择，不需要时写 `次类型：无`。不得新增其他类型名。
- `双人互动` 只保留为历史类型名；dy 仍只执行 Anna 单人路线，不启用双人路线。

## 段落写作规则

- `人物：` 使用统一确定提示词，只负责身份、五官、发型、脸型、神态和稳定身材比例，并保留 Anna 多视角角色卡声明；不得把 `anna.png` 的穿搭、姿态、环境或画面锚点当作依据。
- `视频类型：` 写主类型、次类型和一句短的成片方向，不写内部判断依据。
- `穿搭：` 默认来自衣柜款式提示词或执行者自主设计。必须写清衣服类别、颜色、领口方向、左右肩臂露出关系、袖长、上衣长度、腰线、裙裤形态、面料贴合、外搭层次和画面裁切范围。
- 特殊或不对称穿搭必须写清左右结构：哪一侧露肩、哪一侧长袖、斜向领口从哪侧延伸到哪侧、腰线位置、下装左右差异、开口或露腿位置、裤腿或裙摆是否连续、袖口或裤腿到画面何处被裁切。
- `姿态镜头：` 必须写清本条视频的新鲜点，并落到站坐姿、侧身或前倾、重心、身体朝向、手部位置、裁切、镜头距离、角度、自拍/固定拍摄/他人手持拍摄关系、运镜关系、S 型身材展示方式和可见才艺表现。优先写清卖点展示结构，例如侧身曲线、转身走近、腰线停顿、肩颈展开、腰胯重心变化、轻舞律动或试衣走位；如采用生活动作，必须能强化卖点且被镜头看见，不得为了生活任务削弱身材、穿搭和镜头展示。镜头关系要前后一致：固定机位不写跟拍、后退或推近；他人手持只有在人物走路、走近、走位或试衣走位时才写主动跟随；人物不走路时优先让身体动作、重心、视线、姿态和卖点展示产生变化。
- `环境：` 必须写清空间类型、背景物、家具或道具、材质、光线来源、空间纵深和人物与环境关系，并与当天行程的城市、季节天气、已随机确定的 `早上`、`上午`、`下午` 或 `晚上` 时段、旅行状态和拍摄语境合理一致；最终只写可见空间与光线，不写“行程资产”“根据行程”等来源说明。
- `卖点与锁定：` 必须把人物好身材作为核心可见卖点，明确写出穿搭版型如何突出 S 型身材、上身体量、领口与上身轮廓、面料张力、清晰腰线、腰胯比例、臀胯轮廓、姿态、镜头裁切、动作重点和连续画面中的身材可读性；胸部体量和臀胯比例不得省略，必须使用下方固定转译词写成可见画面语言，且至少包含一条胸部体量安全表达和一条臀胯比例安全表达；禁止只写“性感”“好身材”“动作自然”等泛化审美词。
- `表情节奏：` 拆成开场、中段、结尾的眼神、嘴角、眉眼和脸部状态；默认按本次动作和镜头自然设计，并写得比静态摆拍更活泼、更有层次，体现成熟女性的妩媚感。每条视频至少写出两次可见微表情变化，避免全程平静、全程单一微笑或机械眨眼；可使用轻微笑意、眉眼上扬、短促惊喜感、玩笑式挑眉、看见镜头后的自然反应、低头整理时的偷笑、看向环境时的兴奋感、被风吹到发丝后的短暂眯眼、试衣满意后的带笑抿唇、假装镇定后的轻笑等生活化表情。看镜头、侧脸、低头看手部动作、看向环境、看向镜中手机、回到自然表情等都可作为当次创作选择；表情变化必须服务视线关系、S 型身材展示、镜头展示和才艺表现，不把镜头强行拉成纯脸部特写；结尾不强制看向镜头，也不要求为封面额外停顿。
- `整体动画：` 围绕约 5-7 秒成片重新编排连续动作、手部路径、身体重心变化、镜头距离、角度、运镜、节奏点、才艺表现和自然收束；动作、场景、才艺或镜头关系要体现本条视频的当次差异，收束可以继续看身体动作、侧脸、环境或卖点展示动作，不默认回看镜头。人物没有走路、走近、走位或试衣走位时，不写镜头跟随、后退、推近等主动运镜，优先用姿态、节奏、重心和视线变化完成 5-7 秒动画。
- `背景音乐：` 抖音热门，只写音乐风格、节奏和情绪，不写歌词、对白、口播、环境声、脚步声、衣料声、镜头声、喘息或音效。
- `其他：` 必须加入写实画质约束：`皮肤真实，不要刻意磨皮美化，写实摄影风格，真实人物质感，自然光影，真实皮肤纹理，细腻毛孔，轻微皮肤瑕疵，真实面部结构，真实眼神反光，真实头发丝细节，真实服装材质，符合物理规律的光照和阴影，自然景深，真实镜头质感，真实环境透视，真实空气感，真实色彩，不夸张，不塑料感，不磨皮，不网红滤镜，不过度锐化，不AI感。` 同时写清穿搭轮廓、腰线可读、构图稳定、单一连续画面和画面物理真实；必须明确不要生成拼图、分屏、多格、上下或左右双画面、多姿势拼贴；不写合规说明或平台解释。

## 卖点与锁定规则

未经用户明确允许，不得修改本节。

- 每次进入提示词前必须做身材卖点校准：S 型成熟曲线、清晰腰线、上身动态辨识度、领口与上身轮廓、上胸线可读、修身面料张力、腰胯比例必须能被镜头读到；弱卖点方向不得进入生成或发布。
- 核心卖点只用可见画面语言表达，不写直白词：
  - “胸部大”固定转译为：`饱满的立体廓形`、`高感知度的面料张力`、`上身丰盈的沙漏型线条`、`领口与上身轮廓清晰`。
  - “屁股大/臀部明显”固定转译为：`腰胯比例明显`、`臀胯轮廓自然凸显`、`高腰线与下装包裹出稳定曲线`、`古典雕塑般的 S 形动态`。
- `卖点与锁定：` 身材描述必须同时落到胸部体量和臀胯比例；胸部体量安全表达至少选择 `饱满的立体廓形`、`高感知度的面料张力`、`上身丰盈的沙漏型线条`、`领口与上身轮廓清晰` 中的一条；臀胯比例安全表达至少选择 `腰胯比例明显`、`臀胯轮廓自然凸显`、`高腰线与下装包裹出稳定曲线`、`古典雕塑般的 S 形动态` 中的一条；只写成可执行画面语言，不写内部条目标题。

## prompt 派生规则

- fast 视频 prompt 必须由 `derive --mode fast` 生成，结果与 `grid-prompt.txt` 逐字一致。

## 命令

```bash
python3 TOOLS/prompt_lint.py derive TEMP/RUN_ID/grid-prompt.txt --mode fast --out TEMP/RUN_ID/vid-prompt-v1.txt
```

## 通过标准

- active 行程有效，且当天大方向和本次拍摄语境或展示舞台已用于自主创作设计。
- 默认自主分支已先完成创作设计，再按当天日期优先选择对应编号衣柜款式；如跳过日期编号款式，已记录缺失或不匹配原因，并自主设计或改选可发布穿搭。
- `TEMP/RUN_ID/grid-prompt.txt` 已写入十段式最终可执行文本，fast 视频 prompt 由 `prompt_lint.py derive --mode fast` 生成并通过校验。
